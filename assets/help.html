Cuttly

How to get Api Key?

  1. Go to cutt.ly
  2. Register a Account
  3. Go to cutt.ly/edit and copy the Api Key
  4. Paste the Api Key in Settings

How to Use?

  1. Copy the link to be Shortened to clipboard
  2. Click AutoShare Button

What does AutoShare button do?

  • Fetches the link from clipboard, Shortens and then invokes Share
